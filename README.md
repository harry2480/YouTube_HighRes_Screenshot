# YouTube High-Res Screenshot

YouTube動画の高解像度スクリーンショットを、UI要素なしで簡単に撮影できるChrome拡張機能です。

## 特徴

- **高解像度キャプチャ**: 動画のネイティブ解像度でスクリーンショットを保存します。
- **クリーンな画像**: 再生バー、ボタン、字幕などのUI要素を含まず、純粋な動画フレームのみを保存します。
- **ファイル名自動生成**: 動画のタイトルと再生位置（タイムスタンプ）を含んだファイル名で保存されます。

## 使い方

以下の3つの方法でスクリーンショットを撮影できます：

1. **プレーヤー上のボタン**: 動画プレーヤーの右下（設定アイコンの近く）に追加されるカメラアイコンをクリックします。
2. **ショートカットキー**: キーボードの `S` キーを押します（入力フォームへの入力中は無効）。
3. **拡張機能アイコン**: ブラウザのツールバーにある拡張機能アイコンをクリックし、「Take Screenshot」ボタンを押します。

## インストール方法

この拡張機能は現在開発者モードでのインストールが必要です。

1. このリポジトリをクローンまたはダウンロードします。
2. 必要な依存関係をインストールし、ビルドします。

   ```bash
   npm install
   npm run build
   ```

3. Chromeブラウザを開き、アドレスバーに `chrome://extensions` と入力します。
4. 右上の「デベロッパー モード」スイッチをオンにします。
5. 「パッケージ化されていない拡張機能を読み込む」ボタンをクリックします。
6. プロジェクトフォルダ内の `dist` フォルダを選択します。

## 開発

### 必要要件

- Node.js (v18以上推奨)
- npm

### コマンド

- **開発サーバー起動**:

  ```bash
  npm run dev
  ```

  ファイルの変更を監視し、自動的に再ビルドします。

- **本番ビルド**:

  ```bash
  npm run build
  ```

  `dist` フォルダに配布用のファイルを生成します。

- **テスト実行**:

  ```bash
  npm run test
  ```
